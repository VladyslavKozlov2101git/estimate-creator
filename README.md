# Estimate Creator (Генератор ТЗ)

## Опис проекту

**Estimate Creator** — це внутрішній інструмент для PM та Sales-менеджерів, який дозволяє швидко структурувати вимоги клієнта після созвонів та генерувати технічне завдання (ТЗ) у форматі JSON для розробників.

## Цільова аудиторія

- **Project Managers (PM)**
- **Sales Managers**

## Основний сценарій використання

Користувач проводить зустріч із замовником, заповнює поля форми (назва, вимоги, стек), і в реальному часі бачить сформований JSON, який потім можна передати команді або використати для подальшої автоматизації.

## Функціональні вимоги

### 1. Інтерфейс (Split Screen)

Інтерфейс розділений на дві основні частини:

- **Ліва частина:** Інтерактивна форма для введення даних.
- **Права частина:** Live-прев'ю згенерованого JSON-об'єкта.

### 2. Форма введення (Ліва панель)

Повинна містити наступні типи елементів:

- **Inputs (текстові поля):**
  - Назва проекту (обов'язкове).
  - Замовник / Компанія (обов'язкове).
  - Бюджет (опційно).
- **Textareas (багаторядкові поля):**
  - Короткий опис проекту.
  - Технічні обмеження або особливі побажання.
- **Checkboxes (вибір функцій/модулів):**
  - Авторизація (JWT, OAuth).
  - Платіжна система (Stripe, PayPal).
  - Адмін-панель.
  - Мобільна адаптація.
- **Select / Multi-select:**
  - Стек технологій (React, Node.js, Python, і т.д.).

### 3. JSON Preview (Права панель)

- Дані у правій панелі повинні оновлюватися миттєво при зміні будь-якого поля у формі.
- JSON має бути відформатований (з відступами).
- Кнопка "Copy to Clipboard" для швидкого копіювання результату.

## Технічний стек (Пропозиція)

- **Frontend:** Vite + React (або Vanilla JS для легкості).
- **Styling:** Modern CSS (Flexbox/Grid), Glassmorphism ефекти для преміального вигляду.
- **Library:** `react-hook-form` (якщо React) для зручної роботи з формами.

## Дизайн та Естетика

- **Стиль:** Мінімалістичний, сучасний "Saas-style" інтерфейс.
- **Тема:** Підтримка Dark/Light mode.
- **Акценти:** Виразні стани фокусу полів, плавні анімації появи JSON.
